        # FlowPing
     
        Developed by **Hassan Naseer Ahmed**
     
        A standalone outreach + email-ops app to manage leads, templates, and campaigns.
     
        ### Tech Stack
        - **Backend:** Node.js + Express
        - - **Storage:** local JSON files (no DB required)
          - - **Frontend:** React + Vite
           
            - ### Features
            - - Lead management (clinics)
              - - Email templates
                - - Campaign sequences
                  - - SMTP sending (with dry-run mode)
                    - - Activity logs
                     
                      - ### Run (dev)
                      - In two terminals:
                      - ```
                        cd server && npm install && npm run dev
                        cd web && npm install && npm run dev
                        ```
